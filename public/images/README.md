# Image Placeholders

This directory contains placeholder images for the Disaster Response Predictor application.

## Placeholder Images

- `dashboard-view.png` - Dashboard screenshot placeholder
- `risk-graph-placeholder.png` - Risk graph visualization placeholder
- `weather-map.png` - Weather map placeholder
- `architecture-diagram.png` - System architecture diagram placeholder

## Usage

Replace these placeholders with actual screenshots and diagrams for your project presentation.

## Recommended Image Sizes

- Dashboard screenshots: 1920x1080px
- Graphs/Charts: 800x600px
- Architecture diagrams: 1200x800px
- Icons: 512x512px

## Image Formats

Use PNG for screenshots and diagrams for best quality.
Use SVG for icons and logos for scalability.
